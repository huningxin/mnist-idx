# mnist-idx

## Install

```sh
> npm install
```

## Usage

```sh
> node index.js --help
Usage: index [options]

Options:
  -V, --version        output the version number
  -d, --digit <digit>  The digit to generate
  -h, --help           display help for command
```

For example
```sh
> node index.js -d 1
Wrote digit 1 data to 1.idx
```
